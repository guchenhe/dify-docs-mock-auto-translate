# Frequently Asked Questions

Common questions and answers about using Dify.

## General Questions

### What is Dify?

Dify is an open-source platform for building AI workflows. It allows you to visually design and deploy AI applications without extensive coding.

### Is Dify free to use?

Yes, Dify is open-source and free to use. You can self-host it or use the cloud version.

## Technical Questions

### What programming languages does Dify support?

Dify provides APIs that can be called from any programming language. The platform itself is built with Python and TypeScript.

### Can I integrate my own AI models?

Yes, Dify supports integration with various AI model providers including OpenAI, Anthropic, and custom models.
