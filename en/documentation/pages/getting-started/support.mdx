# Getting Support

How to get help with Dify.

## Community Support

Join our active community:
- **Discord**: Real-time chat with the community
- **GitHub Discussions**: Long-form questions and answers
- **Forum**: Community-driven support

## Documentation

Browse our comprehensive documentation for guides and tutorials.

## Enterprise Support

Contact our sales team for enterprise support options:
- **24/7 Priority Support**: Get help anytime
- **Dedicated Account Manager**: Personalized assistance
- **SLA Guarantees**: Committed response times
- **Custom Training**: Onboarding for your team

## Reporting Bugs

Found a bug? Help us improve:
1. Check if it's already reported on GitHub Issues
2. Provide detailed reproduction steps
3. Include system information and logs
4. Add screenshots if applicable

## Feature Requests

Have an idea? We'd love to hear it:
- Submit via GitHub Discussions
- Vote on existing requests
- Participate in community feedback sessions
