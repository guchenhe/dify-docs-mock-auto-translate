Get started with Dify in minutes.

## Prerequisites

- Docker installed
- 4GB+ RAM available

## Installation

```bash
git clone https://github.com/langgenius/dify.git
cd dify
docker compose up -d
```

Access Dify at `http://localhost:3000`
