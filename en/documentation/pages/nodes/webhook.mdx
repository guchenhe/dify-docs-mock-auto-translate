# Webhook Node

Receive HTTP webhooks in your workflow.

## Overview

The Webhook node allows your workflow to be triggered by external HTTP requests.

## Configuration

- **Endpoint**: Auto-generated webhook URL
- **Authentication**: Optional API key validation
- **Payload**: Access webhook data in your workflow

## Usage

Use this node to:
- Receive data from external systems
- Trigger workflows from other applications
- Build event-driven integrations
- Accept form submissions

## Example

Create a webhook that processes incoming data and triggers your AI workflow.
