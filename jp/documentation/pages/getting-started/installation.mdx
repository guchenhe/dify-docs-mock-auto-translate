独自のインフラストラクチャにDifyをインストールします。

## Docker Compose

Docker Composeでクイックセットアップ：

```bash
git clone https://github.com/langgenius/dify.git
cd dify/docker
docker compose up -d
```

`http://localhost:3000`でアクセスしてください
