# 用語集

Difyの主要な用語と定義。

## A

**エージェント**: 指示と利用可能なツールに基づいて意思決定を行い、アクションを実行できる自律的なAIコンポーネント。

**API Key**: DifyのAPIへのリクエストを認証するために使用される一意の識別子。

## D

**Dify**: AIワークフローとアプリケーションを構築するためのオープンソースプラットフォーム。

## L

**LLM (Large Language Model)**: 膨大な量のテキストデータで訓練されたAIモデルの一種で、人間らしいテキストを理解し生成することができる。

## N

**ノード**: 特定の機能を実行するDifyワークフロー内の単一のステップまたはコンポーネント。

## W

**ワークフロー**: AIアプリケーション内でデータがどのように流れ、処理されるかを定義する接続されたノードのシーケンス。

**WebApp**: ユーザーがブラウザを通じてアクセスできるDifyアプリケーション用のWebベースのインターフェース。
