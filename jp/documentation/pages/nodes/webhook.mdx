# Webhookノード

ワークフロー内でHTTP webhookを受信します。

## 概要

Webhookノードを使用すると、外部のHTTPリクエストによってワークフローをトリガーできます。

## 設定

- **エンドポイント**: 自動生成されたwebhook URL
- **認証**: オプションのAPIキー検証
- **ペイロード**: ワークフロー内でwebhookデータにアクセス

## 使用方法

このノードを使用して以下を実行できます：
- 外部システムからデータを受信
- 他のアプリケーションからワークフローをトリガー
- イベント駆動型の統合を構築
- フォーム送信を受け付け
