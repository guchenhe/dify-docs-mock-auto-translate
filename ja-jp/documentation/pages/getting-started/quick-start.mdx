Difyの真の価値は、どれほど複雑であっても、AIアイデアを簡単に構築、デプロイ、スケールできることにあります。Difyは迅速なプロトタイピング、スムーズな反復、そしてあらゆるレベルでの信頼性の高い本番デプロイメントのために構築されています。

まず、信頼性の高い大規模言語モデル統合をあなたのアプリケーションに組み込むことから始めましょう。この包括的なガイドでは、ユーザーの質問を分類し、大規模言語モデルを使用して直接応答し、興味深い国固有の豆知識でレスポンスを強化するシンプルながら強力なチャットボットを構築します。

## ステップ1: 新しいワークフローを作成する（2分）

1. **Studio** > **ワークフロー** > **空白から作成** > **オーケストレート** > **新しいチャットフロー** > **作成** に移動します

## ステップ2: ワークフローノードを追加する（6分）

<Tip>
  変数を参照したい場合は、最初に `{` または `/` を入力すると、ワークフローで使用可能な異なる変数を確認できます。
</Tip>

### 1. 大規模言語モデルノードと出力: 質問を理解し回答する

<Info>
  `大規模言語モデル` ノードは、ユーザー入力に基づいて応答を生成するため、大規模言語モデルにプロンプトを送信します。API呼び出し、レート制限、インフラストラクチャの複雑さを抽象化するため、ロジックの設計に集中できます。
</Info>

<Steps>
  <Step title="大規模言語モデルノードを作成">
    `ノードを追加` ボタンを使用して大規模言語モデルノードを作成し、開始ノードに接続します
  </Step>
  
  <Step title="モデルを設定">
    デフォルトモデルを選択します
  </Step>
  
  <Step title="システムプロンプトを設定">
    これをシステムプロンプトフィールドに貼り付けます:

    ```text
    The user will ask a question about a country. The question is {{sys.query}}  
    Tasks: 
    1. Identify the country mentioned. 
    2. Rephrase the question clearly. 
    3. Answer the question using general knowledge. 
    
    Respond in the following JSON format: 
    {   
      "country": "<country name>",
      "question": "<rephrased question>",
      "answer": "<direct answer to the question>" 
    }
    ```
  </Step>
  
  <Step title="構造化出力を有効にする">
    **構造化出力を有効にする** により、大規模言語モデルが返す内容を簡単に制御し、正確なデータ抽出や条件ロジックでの下流使用のための一貫性のある機械可読出力を確保できます。
    
    - 出力変数構造化をオンにトグル > `設定` をクリックし、`JSONからインポート` をクリック
    - 貼り付け:
    
    ```json
    ": "string" 
    }
    ```
  </Step>
</Steps>

### 2. コードブロック: 豆知識を取得

<Info>
  `コード` ノードはコードを使用してカスタムロジックを実行します。ビジュアルワークフロー内で必要な場所にコードを正確に注入でき、バックエンド全体を配線する手間を省けます。
</Info>

<Steps>
  <Step title="コードノードを作成">
    `ノードを追加` ボタンを使用して `コード` ノードを作成し、大規模言語モデルブロックに接続します
  </Step>
  
  <Step title="入力変数を設定">
    一つの `入力変数` 名を「country」に変更し、変数を `structured_output` > `country` に設定します
  </Step>
  
  <Step title="Pythonコードを追加">
    このコードを `PYTHON3` に貼り付けます:

    ```python
    def main(country: str) -> dict:     
      country_name = country.lower()     
      fun_facts = {
        "japan": "Japan has more than 5 million vending machines.",
        "france": "France is the most visited country in the world.",
        "italy": "Italy has more UNESCO World Heritage sites than any other country."     
      }     
      fun_fact = fun_facts.get(country_name, f"No fun fact available for {country.title()}.")
      return {"fun_fact": fun_fact}
    ```
  </Step>
  
  <Step title="出力変数の名前を変更">
    より良くラベル付けされた変数を持つため、出力変数 `result` を `fun_fact` に変更します
  </Step>
</Steps>

### 3. 回答ノード: ユーザーへの最終回答

<Info>
  `回答` ノードは返すためのきれいな最終出力を作成します。
</Info>

<Steps>
  <Step title="回答ノードを作成">
    `ノードを追加` ボタンを使用して `回答` ノードを作成します
  </Step>
  
  <Step title="回答フィールドを設定">
    回答フィールドに貼り付けます:

    ```text
    Q: {{ structured_output.question }} 

    A: {{ structured_output.answer }} 

    Fun Fact: {{ fun_fact }}
    ```
  </Step>
</Steps>
