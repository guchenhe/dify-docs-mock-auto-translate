Difyの真の価値は、どれほど複雑であっても、AIアイデアを簡単に構築、デプロイ、スケールできることにあります。迅速なプロトタイピング、スムーズな反復、そしてあらゆるレベルでの信頼性の高い本番デプロイメント用に構築されています。

アプリケーションへの信頼性の高い大規模言語モデル統合を学ぶことから始めましょう。このガイドでは、ユーザーの質問を分類し、大規模言語モデルを使用して直接応答し、国固有の豆知識で回答を強化するシンプルなチャットボットを構築します。

## ステップ1：新しいワークフローを作成（2分）

1. **Studio** > **ワークフロー** > **空から作成** > **Orchestrate** > **新しいチャットフロー** > **作成** に移動

## ステップ2：ワークフローノードを追加（6分）

<Tip>
  変数を参照したい場合は、最初に `{` または `/` を入力すると、ワークフローで利用可能なさまざまな変数を確認できます。
</Tip>

### 1. 大規模言語モデルノードと出力：質問を理解して回答

<Info>
  `大規模言語モデル`ノードは、ユーザー入力に基づいて応答を生成するために、言語モデルにプロンプトを送信します。API呼び出し、レート制限、インフラストラクチャの複雑さを抽象化するため、ロジックの設計に集中できます。
</Info>

<Steps>
  <Step title="大規模言語モデルノードを作成">
    `ノード追加`ボタンを使用して大規模言語モデルノードを作成し、スタートノードに接続します
  </Step>
  
  <Step title="モデルを設定">
    デフォルトモデルを選択します
  </Step>
  
  <Step title="システムプロンプトを設定">
    これをシステムプロンプトフィールドに貼り付けます：

    ```text
    The user will ask a question about a country. The question is {{sys.query}}  
    Tasks: 
    1. Identify the country mentioned. 
    2. Rephrase the question clearly. 
    3. Answer the question using general knowledge. 
    
    Respond in the following JSON format: "question": "<rephrased question>",
      "answer": "<direct answer to the question>" 
    }
    ```
  </Step>
  
  <Step title="構造化出力を有効化">
    **構造化出力を有効化**により、大規模言語モデルが返すものを簡単に制御でき、正確なデータ抽出や条件付きロジックでのダウンストリーム使用のために一貫性のある機械可読な出力を確保できます。
    
    - 出力変数構造化をONに切り替え > `設定` をクリックし、`JSONからインポート` をクリック
    - 貼り付け：
    
    ```json
    {   
      "country": "string",   
      "question": "string",   
      "answer": "string" 
    }
    ```
  </Step>
</Steps>

### 2. コードブロック：豆知識を取得

<Info>
  `Code`ノードは、コードを使用してカスタムロジックを実行します。ビジュアルワークフロー内で必要な箇所にコードを正確に注入できるため、バックエンド全体を配線する手間を省けます。
</Info>

<Steps>
  <Step title="コードノードを作成">
    `ノード追加`ボタンを使用して`Code`ノードを作成し、大規模言語モデルブロックに接続します
  </Step>
  
  <Step title="入力変数を設定">
    1つの`入力変数`の名前を"country"に変更し、変数を`structured_output` > `country`に設定します
  </Step>
  
  <Step title="Pythonコードを追加">
    このコードを`PYTHON3`に貼り付けます：

    ```python
    def main(country: str) -> dict:     
      country_name = country.lower()     
      fun_facts = {
        "japan": "Japan has more than 5 million vending machines.",
        "france": "France is the most visited country in the world.",
        "italy": "Italy has more UNESCO World Heritage sites than any other country."     
      }     
      fun_fact = fun_facts.get(country_name, f"No fun fact available for {country.title()}.")
      return {"fun_fact": fun_fact}
    ```
  </Step>
  
  <Step title="出力変数名を変更">
    出力変数`result`を`fun_fact`に変更して、より分かりやすくラベル付けされた変数にします
  </Step>
</Steps>

### 3. 回答ノード：ユーザーへの最終回答

<Info>
  `Answer`ノードは、返すためのクリーンな最終出力を作成します。
</Info>

<Steps>
  <Step title="回答ノードを作成">
    `ノード追加`ボタンを使用して`Answer`ノードを作成します
  </Step>
  
  <Step title="回答フィールドを設定">
    回答フィールドに貼り付けます：

    ```text
    Q: {{ structured_output.question }} 

    A: {{ structured_output.answer }} 

    Fun Fact: {{ fun_fact }}
    ```
  </Step>
</Steps>
