# Webhook 节点

在您的工作流中接收 HTTP webhook。

## 概述

Webhook 节点允许您的工作流通过外部 HTTP 请求触发。

## 配置

- **端点**：自动生成的 webhook URL
- **身份验证**：可选的 API 密钥验证
- **负载**：在您的工作流中访问 webhook 数据

## 使用方法

使用此节点可以：
- 从外部系统接收数据
- 从其他应用程序触发工作流
- 构建事件驱动的集成
- 接受表单提交
