# 故障排除指南

本指南帮助您解决在使用 Dify 时遇到的常见问题。

## 常见问题

### 安装问题

如果您在安装过程中遇到问题：

1. **检查系统要求** - 确保您的系统满足最低要求
2. **验证依赖项** - 确保所有依赖项都已正确安装
3. **查看日志** - 检查安装日志以获取具体的错误信息

### 运行时错误

常见的运行时错误及其解决方案：

- **API 连接失败**：检查您的 API 密钥和网络连接
- **模型未找到**：验证模型名称并确保它在您的地区可用
- **超出速率限制**：在您的应用程序中实现适当的速率限制
