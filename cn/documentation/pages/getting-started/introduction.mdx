Dify 是一个用于构建智能体工作流的开源平台。它让您可以可视化地定义流程，连接现有的工具和数据源，并部署解决实际问题的 AI 应用程序。

## 为什么选择 Dify？

**可视化工作流构建器**：通过直观的拖拽界面设计复杂的 AI 工作流。基本用例无需编码。

**企业就绪**：专为规模化构建，具备安全性、合规性和部署选项，满足企业要求。

**可扩展平台**：通过我们全面的插件系统，轻松与您现有的工具、数据库和 API 集成。
