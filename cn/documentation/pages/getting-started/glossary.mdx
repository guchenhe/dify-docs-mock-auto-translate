# 术语表

Dify 的关键术语和定义。

## A

**智能体**：一个自主的 AI 组件，能够基于其指令和可用工具做出决策并采取行动。

**API Key**：用于验证对 Dify API 请求的唯一标识符。

## D

**Dify**：一个用于构建 AI 工作流和应用程序的开源平台。

## L

**LLM（大语言模型）**：一种在大量文本数据上训练的 AI 模型类型，能够理解和生成类人文本。

## N

**节点**：Dify 工作流中执行特定功能的单个步骤或组件。

## W

**工作流**：一系列连接的节点，定义数据在您的 AI 应用程序中如何流动和处理。

**WebApp**：用户可以通过浏览器访问的 Dify 应用程序的基于网页的界面。
