# 手动翻译示例

这个文件是手动创建的中文翻译，用于测试翻译工作流。

## 目的

展示如何手动添加翻译内容：
- 不通过自动翻译系统
- 直接创建翻译文件
- 用于改进现有翻译

## 使用场景

手动翻译适用于：
- 修正自动翻译的错误
- 添加本地化内容
- 优化术语翻译
- 改善表达方式

## 工作流

手动翻译的工作流程：
1. 创建新的翻译文件
2. 提交 PR（仅包含翻译文件）
3. 通过人工审核
4. 合并到主分支

这种方式不会触发自动翻译工作流。
